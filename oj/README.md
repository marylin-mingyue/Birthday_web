# OJ 版：二叉树遍历（stdin/stdout）

这个目录提供一个**真正可以丢到 Online Judge** 跑的版本（不依赖网页）。

## 输入格式（自定义但常见）

两行：

1) `order`：`pre` / `in` / `post`（分别表示前序/中序/后序）

2) 层序（level-order）节点序列，用空格分隔；空节点用 `#` 表示。节点值按**字符串 token** 读取（所以不要在值里写空格）。

示例：

```
pre
温柔 靠谱 幽默 耐心 上进 担当 细心 # # # # # # # #
```

## 输出格式

一行，按指定遍历顺序输出所有非空节点值，用空格分隔。

## 本地运行

```bash
c++ -std=c++17 -O2 main.cpp -o main
printf "in\nA B C # D # #\n" | ./main
```
